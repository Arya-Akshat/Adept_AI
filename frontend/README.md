# ADEPT - Adaptive Education Platform

ADEPT is an AI-powered study companion that transforms your syllabus and study materials into personalized learning roadmaps.

## Features
-   **PDF Upload**: Upload your study materials (notes, books, syllabus).
-   **AI Roadmaps**: Automatically generate structured study roadmaps.
-   **Topic Explanations**: Get detailed explanations for every topic using Gemini AI.
-   **Curated Resources**: Find relevant YouTube videos for each topic.

## Tech Stack
-   **Frontend**: React, Vite, TypeScript, Tailwind CSS, shadcn/ui
-   **Backend**: Node.js (Express), Python (Flask)
-   **AI**: Google Gemini 2.0 Flash
-   **Database**: Local JSON storage (MVP) / MongoDB (Planned)

## Getting Started

1.  **Install Dependencies**
    ```bash
    cd frontend
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```
